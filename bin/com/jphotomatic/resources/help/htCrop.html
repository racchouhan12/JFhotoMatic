HOW TO CROP AN IMAGE?
To crop an image:-
  • Load an image from the FILE menu i.e Import image.
  • Specify the x,y, width & height for the new image in the textboxes of the com.jphotomatic.shape.Crop tab.
  • Finally press com.jphotomatic.shape.Crop button.
The x,y,width & height specified will be the Dimension/Size of the new image.
