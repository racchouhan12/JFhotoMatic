HOW TO GIVE COLOR EFFECTS TO AN IMAGE?
To give color effects to an image just follow the points given below:-
  • Load an image from the FILE menu i.e Import image.
  • Then press Red Band Effect button of the Color Effects tab or select Red Effect.. from Edit menu to give red band to the image.
    • OR
  • Press Green Band Effect button of the Color Effects tab or select Green Effect.. from Edit menu to give green band to the image.
    • OR
  • Press Blue Band Effect button of the Color Effects tab or select Blue Effect.. from Edit menu to give blue band to the image.
    • OR
  • Press Yellow Band Effect button of the Color Effects tab or select Yellow Effect.. from Edit menu to give yellow band to the image.
